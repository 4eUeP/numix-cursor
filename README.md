numix-cursor
============

An X-cursor-theme for use along with the numixproject products.


## Installing

Dependencies:

- inkscape
- xcursorgen

Run the following commands:

```sh
make build

mkdir -p ~/.icons/Numix-Cursor  # or /usr/share/icons for all users
cp -R ./build/dist/* ~/.icons/Numix-Cursor/
```
